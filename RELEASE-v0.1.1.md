# Release v0.1.1 — The Counting Agents

**Date:** 2026-03-03

A documentation and polish release that establishes English as the primary language, adds full German translations, introduces a guide for custom OpenCode agents, and refines the tmux session layout.

---

## Improvements

- **English-first documentation** — README, scripts documentation, and all primary docs are now consistently written in English.
- **Improved tmux layout** — The session layout has been reorganized for better readability: Counter now occupies the top-left two-thirds with Control beside it, while Odd, Even, and Prime share the bottom row equally.

## Documentation

- **German translations** — Full German versions of the README (`readme_de.md`), experiment documentation (`docs/experiment_de.md`), and scripts guide (`docs/scripts_de.md`) are now available.
- **OpenCode custom agents guide** — New documentation (`docs/opencode-custom-agents.md`) explains how to configure and use the counting agents as custom agents inside OpenCode, available in both English and German.
